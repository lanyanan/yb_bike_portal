# bike_portal

## 目录结构  
本项目基于 [vuejs](https://vuejs.org), 目录结构由 [vue-cli](https://github.com/vuejs/vue-cli) 生成, 组件库使用 [iview](https://www.iviewui.com/)

## 开发

``` bash
# node >= v7.7.4

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```

## 构建 
```
npm run build

```
